Scores
======

## Usage

```
# Using a file
./bin/scores file_name.txt
# Using STDIN
cat file_name.txt | ./bin/scores
```

## Running the tests

The tests can be run using `rspec`.

## Dependencies

The target version of Ruby is 2.5.0.

There are no runtime dependencies, however in order to run the test suite
rspec is required.

```
$ gem install rspec
```
